=================================================================================
This will eventually become a package to serve/assist with pDNS. For now only does:

1. Distilled log filtering

=================================================================================

Following functionality are provided by the script
--------------------------------------------------
::
        1) Filtering dns logs to be able to exported to pDNS database
		- ts, id.resp_h, query, RA, answers, TTLs
	2) Allows for limiting filtered logs to Recursive Queries only
	3) Allows to geneate logs in tsv or json

Installation
------------
	zeek-pkg install zeek/initconf/pDNS
	or
	@load pDNS/scripts
